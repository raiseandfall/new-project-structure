# New Project Structure
> A simple CL tool to create a new project structure

## CLI

```shell
$ new project-test
```

```shell
$ new --help

  Usage
    $ new <project-name>

  Example
    $ new project-test
```

## CONTRIBUTE

```shell
$ git clone git@github.com:raiseandfall/new-project-structure.git && cd new-project-structure
```

```shell
$ npm i
```

Run
```shell
$ npm run dev
```

Run tests
```shell
$ npm run test
```
